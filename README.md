# 🧇 WaffleStore - Online Food Ordering Web Application

## 📌 Overview
WaffleStore is a dynamic web-based application developed for ordering food items like waffles, pancakes, desserts, and shakes. It allows users to browse items, add them to cart, and place orders online.

---

## 🎯 Objective
To build a full-stack web application using Java web technologies with database connectivity and session management.

---

## 🚀 Features
- 🛒 Add to Cart functionality
- 🍽️ Browse menu (Waffles, Pancakes, Desserts, Shakes)
- 💳 Checkout and payment options
- 👤 User session tracking
- 📊 Admin dashboard (manage items/orders)
- 💬 User reviews and ratings

---

## 🧰 Technologies Used
- Java (JSP & Servlets)
- HTML, CSS
- JavaScript (optional)
- MySQL (Database)
- Apache Tomcat Server

---

## 🗄️ Database
The application uses MySQL to store:
- Product details
- User orders
- Customer information
- Reviews and ratings

---

## ⚙️ Modules

### 👨‍🍳 User Module
- View menu
- Add items to cart
- Place orders

### 🛠️ Admin Module
- Manage products
- View orders
- Update menu

---

## 🔄 How It Works
1. User selects food items
2. Adds items to cart
3. Proceeds to checkout
4. Chooses payment method
5. Order is stored in database

---

## ▶️ How to Run

1. Install:
   - Java JDK
   - Apache Tomcat
   - MySQL

2. Import project into IDE (Eclipse / VS Code)

3. Configure database connection

4. Deploy project on Tomcat server

5. Open browser:http://localhost:8080/WaffleStore
